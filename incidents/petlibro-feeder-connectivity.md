# Petlibro Feeder Connectivity Incident

## Problem
Petlibro feeder failed to connect and respond to remote commands.

## Impact
Loss of remote control and monitoring functionality for feeding system.

## Symptoms
- Device powered on  
- Application reported device as offline  
- No response to commands  

## Investigation
- Verified power supply was stable  
- Checked Wi-Fi connectivity  
- Confirmed router and network were operational  
- Attempted device re-pairing  

## Root Cause
Likely network or cloud-side connectivity issue preventing device from reaching backend service.

## Resolution
- Reset network connection  
- Reconnected feeder to Wi-Fi  
- Re-established communication with cloud service  

## Verification
- Device appeared online in application  
- Successfully responded to remote commands  

## Follow-up
Prioritise checking network and cloud reachability early in future connectivity-related incidents.
